Je kunt Alpino in Docker op twee manieren starten. 

De eerste manier is met alleen een directory als argument, bijvoorbeeld:

[36m    alpino.bash $HOME/alpino[0m

Je komt dan in een shell waarin je programma's kunt uitvoeren.

(op een ouder Windows-systeem gebruik je [4malpino.cmd[0m i.p.v. alpino.bash)

De tweede manier is met na de directory een programma-opdracht, eventueel
gevolgd door argumenten, bijvoorbeeld:

[36m    alpino.bash $HOME/alpino ls -la /work[0m

In dit geval wordt het programma [4mls[0m direct gestart, en na het
beÃ«indigen stopt Alpino in Docker.

----

De directory die je opgeeft als eerste argument is de verbinding met je
data buiten Docker. Deze directory wordt binnen Docker gekoppeld aan de
directory [4m/work/data[0m . Wanneer je in de shell in Docker zit dan is deze
directory ook beschikbaar als [4m~/data[0m .

----

[36m    alpino.bash -u[0m
