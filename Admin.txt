
Met deze repo doe je het volgende:

- Een binaire distro van Alpino maken, gecompileerd op Ubuntu/Xenial
    -> alpino/Alpino*.tar.gz
- AppImages maken van alto
    -> work/tools/alto/alto-x86_64.AppImage
    -> work/tools/alto/alto_v6-x86_64.AppImage
- Een image maken van Alpino in Docker, eveneens op basis van
  Ubuntu/Xenial, en die pushen naar de server

Alles gebeurt door 'make'. Run make zonder argumenten voor een lijst van
targets.

Wanneer je iets bijwerkt hoef je gewoonlijk niet alle targets opnieuw te
maken. Als bijvoorbeeld Alpino zelf is veranderd, dan zijn alleen deze
stappen nodig:

    make step1  # je docker image bijwerken, voor alle zekerheid
    make step3  # Alpino compileren (download van github)
    make step8  # image van Alpino in Docker maken
    make step9  # image van Alpino in Docker naar de server pushen

En verder moet je nog alpino/Alpino*.tar.gz op je website zetten

'make' probeert alle geproduceerde bestanden schrijfbaar te houden voor
groep 'software'.
